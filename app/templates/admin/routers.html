What are Smart Routers?

Routers act as intelligent traffic controllers. They examine each request and route it to the appropriate model based on rules you define.

  • Priority: Always tries the first model, falls back to second if unavailable
  • Random: Distributes load evenly across all targets
  • Least Loaded: Routes to the server with fewest active requests (best for high throughput)
