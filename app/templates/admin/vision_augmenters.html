What are Vision Augmenters?

Give text-only models vision capabilities. If a request includes an image, the augmenter offloads it to a Vision Model (VLM) first. The VLM's text description is then combined with the original prompt and sent to your powerful text model.

User: [Image] "What color is this?"
1. VLM: "The image shows a red apple."
2. Text Model: Red apple description + User Query -> "The apple is red."
